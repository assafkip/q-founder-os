# Decision Log

> Active rules governing system behavior. Referenced during morning routine and pipeline management.

## Format

Each decision entry:
```
### [DATE] Decision title
**Origin:** [USER-DIRECTED | CLAUDE-RECOMMENDED -> APPROVED | CLAUDE-RECOMMENDED -> MODIFIED | CLAUDE-RECOMMENDED -> REJECTED | SYSTEM-INFERRED]
**Context:** Why this decision was made
**Rule:** What the system should do going forward
**Expires:** [date or "never"]
```
