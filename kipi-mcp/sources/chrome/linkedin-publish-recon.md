Navigate to your LinkedIn profile, then click Activity, then the Posts tab.

Check if any posts from the last 48 hours match drafted content.

For each post found, extract:
- **platform**: "linkedin"
- **post_url**: Direct URL to the post
- **text**: First 200 characters of the post text
- **published_at**: When the post was published

Then, if an X/Twitter profile is configured, navigate to it and check for posts from the last 48 hours. Extract the same fields with platform set to "x".
