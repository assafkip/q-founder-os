# Content Guardrails

> Every piece of content must pass these checks before publishing.

## PASS/FAIL Checks

### 1. Voice Check
- [ ] Sounds like the founder (not generic marketing copy)
- [ ] Zero banned AI words (see founder-voice skill)
- [ ] No hedging language
- [ ] Sentence length varies

### 2. Positioning Check
- [ ] Does NOT misclassify the product
- [ ] Uses approved language from talk-tracks.md
- [ ] No overclaiming ({{UNVALIDATED}} markers respected)
- [ ] Factual claims backed by canonical files or sources (use `/q-research` for verification)

### 3. Anti-AI Check
- [ ] No "Furthermore/Moreover/Additionally" paragraph openers
- [ ] No uniform sentence length
- [ ] No formulaic closing that restates content
- [ ] Passes the "who wrote this" test

### 4. Platform Check
- [ ] Meets platform length requirements
- [ ] Appropriate hashtags (if applicable)
- [ ] CTA is clear and single

### 5. Sensitivity Check
- [ ] No naming competitors negatively
- [ ] No revealing confidential customer/partner info
- [ ] No claims without evidence
