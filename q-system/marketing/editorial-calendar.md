# Editorial Calendar

> Weekly theme assignments and content slots. Updated by `/q-market-plan`.

## How to Use

1. `/q-market-plan` reads this file to see what's assigned this week
2. After planning, `/q-market-plan` writes the new assignments here
3. `/q-market-create` reads the current week's assignment before generating
4. `/q-market-publish` marks slots as published
