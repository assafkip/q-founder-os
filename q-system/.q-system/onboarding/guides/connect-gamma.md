# Connect Gamma

## What This Does
Gamma generates pitch decks, one-pagers, and visual presentations. When Kipi needs to create a deck or a visual document, Gamma does the design work.

## What the User Needs
- A Gamma account (free tier available at gamma.app)

## Walk-Through Script

### Part 1: Explain

Say:

> "Let's connect Gamma - this lets me create slide decks, one-pagers, and presentations for you.
>
> When you ask me to make a pitch deck or a leave-behind document, Gamma handles the design. You get a polished, shareable link.
>
> Gamma has a free tier that lets you create a few presentations. Good enough to get started."

### Part 2: Connect via Claude.ai MCP

If the user is on claude.ai/code, Gamma may be available as a built-in integration:

> "Check if Gamma is available in your integrations:
>
> 1. Look for **'Connect tools'** or **'Integrations'** in your Claude settings
> 2. Search for **Gamma**
> 3. If it's there, click **Connect** and sign in
>
> If you don't see it, we'll skip this for now. You can always create decks later by copying content into gamma.app manually."

### Part 3: Verify

> "Let me test the connection... [run test]. Gamma is connected! I can now generate decks for you."

## When to Recommend This

- For `gtm-founder` (pitch decks, investor updates, sales one-pagers)
- For `content-creator` (visual content, presentation-style posts)
- Always Optional tier - never block setup on this

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "I don't see Gamma in integrations" | "Gamma might not be available as a direct integration yet. No worries - when you need a deck, I'll generate the content and you can paste it into gamma.app to get the design." |
| "I already use Google Slides / Keynote" | "You can keep using those. I'll generate the content and structure, and you can copy it into your preferred tool." |
| "Is Gamma free?" | "Gamma has a free tier with limited presentations. Paid plans start around $10/month if you need more." |
