# Social Engagement Playbook

> Rules for commenting, DMs, and relationship building on social platforms.

## Core Rules

1. **Value-first**: Every comment must add value. No "Great post!" or "+1" comments.
2. **No pitching in comments**: Share expertise, not product links.
3. **Match energy**: If they're casual, be casual. If they're technical, be technical.
4. **One comment per post**: Don't dominate the thread.
5. **Reply to replies**: If someone engages with your comment, always respond.

## Comment Styles

| Style | When to use | Example pattern |
|-------|-------------|-----------------|
| Expert addition | Post in your domain | "We saw this at [company] — [specific insight]" |
| Respectful challenge | Debatable claim | "Interesting take. One thing I'd push back on..." |
| Bridge builder | Connect two ideas | "[Author]'s point connects to [other concept] because..." |
| Data point | Claims need evidence | "The data supports this — [specific stat or observation]" |
| Question | Spark deeper discussion | "Curious about [specific aspect] — have you seen..." |

## DM Rules

1. **Only DM after 2+ comment interactions** (unless warm intro)
2. **Lead with their content**: Reference something specific they posted
3. **Short**: 2-3 sentences max for first DM
4. **No links in first DM**: Build the relationship first
5. **Async-first**: Suggest a demo link or async resource before a call

## Platform-Specific

### LinkedIn
- Comment within 1 hour of post for algorithm boost
- Tag people only when genuinely relevant
- Carousel comments: reference a specific slide

### X/Twitter
- Quote tweets > replies for visibility
- Thread replies should add context, not just agree
