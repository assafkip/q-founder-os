Navigate to https://www.linkedin.com/messaging/

Detect outbound actions from the last 48 hours.

For each outbound action, extract:
- **action_type**: One of: comment, dm_sent, connection_request
- **target**: Name of the person the action was directed at
- **text**: Content of the message or comment
- **timestamp**: When the action occurred

Check sent messages, then navigate to recent post activity to find comments you posted.
